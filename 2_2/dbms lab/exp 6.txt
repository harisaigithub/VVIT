--Database Connectivity:

create or replace procedure connect_to_database (
  username varchar2,
  password varchar2,
  database_name varchar2
) is
  connection varchar2 (32);
begin
  -- connect to the database
  connection := 'oci:dbname=' || database_name || ';charset=AL32UTF8';
  open :connection for
    'connect ' || username || '/' || password || '@' || connection;
  if (sql%notfound) then
    dbms_output.put_line ('Could not connect to database.');
    return;
  end if;
  dbms_output.put_line ('Connected to database successfully.');
end;
/

NOTE:
This procedure takes three parameters: the username, password, and database name. It then connects to the database using the Oracle Call Interface (OCI). If the connection is successful, the procedure prints a message to the console. Otherwise, it prints an error message.

To use this procedure, simply call it with the appropriate parameters. For example, to connect to a database named mydb using the username scott and password tiger, you would call the procedure like this:

Code snippet
connect_to_database ('scott', 'tiger', 'mydb');
